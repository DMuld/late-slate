## Getting Started:
1. `git clone https://github.com/late-slate/late-slate.git`
2. `cd late-slate`
3. `npm install`
4. Copy 'sample.env' to '.env' (Adding your own env variables)
5. Run `docker compose up -d`
6. Run `npx prisma db push`
7. Run `npx prisma db seed`
8. Run 'npm run dev'
